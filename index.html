Erste Schritte

Willkommen zur Boules Turnierverwaltung.
Um ein Turnier durchzuführen, folge diesen Schritten:
  1. Wähle die Teamgröße und trage die Spieler im Anmeldungen-Tab ein
  2. Wähle eine Turnierart und starte die erste Runde im Turniersysteme-Tab
  3. Trage die Spielergebnisse im Laufende Spiele-Tab ein
  4. Starte die nächste Runde im Turniersysteme-Tab, oder beende die Turnierphase und starte eine Neue
    z.B. Vorrunde beenden und ABCD-Turniere starten
  5. Sieh die Platzierung im Platzierungen-Tab ein
  6. Exportiere die Anmeldung, Platzierungen und Spielergebnisse über das Optionen und Einstellungen-Tab
Weitere Informationen sind über die Links im Über...-Tab verfügbar

Warum Chrome?

Für die Darstellung dieses Programmes wird der Chrome Browser empfohlen.

Es werden HTML5-Elemente und CSS3-Selektoren genutzt, die aktuell nur von Chrome standardkonform unterstützt werden. Andere Browser können zu übergroßen Icons, ungewollten Rändern und fehlenden Inhalten führen. Zusätzlich lassen sich die Übersichten unter Verwendung von Firefox, Opera und Safari leider nicht ausdrucken, sondern führen zum Druck von leeren Seiten. Bis Kompatibilität der anderen Browser hergestellt wurde, muss ich leider um die Verwendung des Chrome-Browsers bitten.

Laden und Speichern

Laden

Das Turnier wird aus einer Datei neu geladen. Eventuell laufende Turniere werden überschrieben.

Dieser Browser kann keine lokalen Dateien laden. Aktualisieren sie ihren Browser oder öffnen sie diese Seite in einem moderneren Browser. Manche Hersteller stellen keine Updates für ihre Browser unter einigen Betriebssystemen zur Verfügung (z.B. Apple mit Safari für Windows).

Turnier speichern

Speichert das aktuelle Turnier in einer Datei. Diese kann später wieder geladen werden, um das Turnier wieder herzustellen.

Um das Turnier nach Schließen des Browsers fortzusetzen, muss der Zustand manuell in einer Datei zwischengespeichert oder ein aktueller Browser installiert werden.


Import & Export

Spielernamen für Autovervollständigung aus dem Internet laden.

Dieser Browser kann keine lokalen Dateien laden.

Tabellen-Export

Exportierte csv-Dateien lassen sich bequem mit Office-Programmen wie Excel, LibreOffice und Google Docs öffnen.

Damit die Datei geöffnet werden kann, muss sie mit der .csv-Dateiendung gespeichert werden, beispielsweise als 'boules.csv'.

ACHTUNG: Dieser Browser unterstützt wichtige Standards nicht, die zum Speichern und Laden notwendig sind. Beim Schließen des Browserfensters gehen sämtliche Daten verloren.

Storage

Dieses Programm und alle Eingaben werden im Browser gespeichert, um auch offline und nach Beenden des Browsers fortfahren zu können. Dazu werden aktuelle Turnierstände nach jeder Änderung automatisch gespeichert.

Browserunterstützung

Für diese Software wird auf freie Webstandards gesetzt. Leider werden diese nicht von allen Browsern vollständig unterstützt. Deshalb kann es vorkommen, dass einzelne Funktionen mit einigen Browsern nicht zur Verfügung stehen. Um das nach Möglichkeit zu vermeiden, wird das Programm mit diesen Browsern getestet:

Desktop:

Mobil:

Dieser Browser unterstützt folgende Funktionen nicht:

  • JavaScript
  • HashChange
  • LocalStorage
  • WebWorkers
  • Download Attribute
  • FileReader
  • JSON

Dieser Browser unterstützt alle notwendingen Funktionen.
